# M_usedAuction 중고경매 어플(Xamarin_Andriod)

대학 팀 프로젝트 과제

DB
- Firebase realtime database 이용
- Tree 구조의 테이블로 구성

App 기능
- 경매 기능
- 검색 및 카테고리 기능
- top 10 제품 보기
- 알림 기능
- 즐겨 찾기
- 평점 및 신뢰도

추가사항
- usedAuction 과 같은 DB사용 윈도우 어플에서도 확인가능
- https://github.com/sweethor/usedAuction
